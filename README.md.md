Servidor MCP operativo para Microsoft Fabric, creado desde cero.

Incluye herramientas para:
- Lakehouse
- Warehouse
- Pipelines
- Notebooks
- Real-Time Intelligence

Este repositorio está diseñado para integrarse con:
- Azure DevOps MCP
- Power BI MCP
- Fabric RTI MCP
- Arquitectura triad-agent (Extractor–Verifier–Publisher)
